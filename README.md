# Installation: 
developed with: 
node.js 10.18.0 LTS
```
git clone https://github.com/mmickee1/Helsinki-Innovation2019.git
cd Helsinki-Innovation2019
```

# in node.js command prompt
```
npm install
expo start 
```
# for android (connect phone via usb, or have Android Studio and emulator installed):
```
a 
```
